About This Project

Project Goal

This AI-powered system aims to assist healthcare professionals in early detection of Left Ventricular Hypertrophy (LVH) using multimodal cardiac data analysis including ECG signals, medical imaging (MRI/CT), and clinical parameters.

Technology Stack

Built with Python, TensorFlow, scikit-learn, and Flask. Utilizes advanced machine learning algorithms including XGBoost, Random Forest, SVM, and deep learning CNNs for comprehensive cardiac assessment.

Model Performance

Our ensemble models achieve high accuracy across all modalities: Clinical (89.13%), ECG (82.00%), MRI (81.43%), and CT (78.75%) with dynamic confidence scoring.

Clinical Use

Designed as a clinical decision support tool to complement physician expertise. Provides risk stratification, confidence scores, and detailed analysis reports for informed medical decision-making.

ECG Analysis Model

Electrocardiogram (ECG) Analysis

Best Model: XGBoost - 82.00% Accuracy
What is ECG?

An Electrocardiogram (ECG) is a medical test that records the electrical activity of the heart over time. It's a non-invasive procedure that provides crucial information about heart rhythm, rate, and structure. In LVH detection, ECG signals show characteristic changes in voltage patterns that indicate enlarged heart muscle.

Advanced Signal Processing Features
  • R-Peak Detection: Identifies heartbeat peaks for rhythm analysis
  • QRS Complex Analysis: Measures electrical conduction through ventricles
  • Heart Rate Variability: Analyzes beat-to-beat variations
  • Sokolow-Lyon Criteria: Classic LVH voltage measurements
  • ST Segment Analysis: Detects ischemia and strain patterns
  • T-Wave Morphology: Analyzes repolarization abnormalities

MRI Analysis Model

Magnetic Resonance Imaging (MRI) Analysis

Best Model: SVM - 81.43% Accuracy
What is Cardiac MRI?

Cardiac MRI is a sophisticated imaging technique that uses magnetic fields and radio waves to create detailed images of the heart. It provides excellent soft tissue contrast, making it the gold standard for measuring heart muscle thickness, chamber volumes, and detecting structural abnormalities associated with LVH.

Advanced Image Analysis Features
  • Wall Thickness Measurement: Precise myocardial wall analysis
  • Chamber Volume Assessment: Left ventricular volume calculations
  • Cardiac Morphology: Shape and structure analysis
  • Texture Analysis: Myocardial tissue characterization
  • Segmentation: Automated heart chamber identification
  • Strain Analysis: Cardiac function assessment

What is Clinical Data Analysis?

Clinical Data Analysis involves processing patient demographics, vital signs, laboratory results, and medical history to identify risk factors and patterns associated with LVH. This approach combines traditional medical knowledge with machine learning to provide comprehensive risk assessment.

Clinical Parameters Analyzed (11 Key Features)
Age: Patient age (1-120 years)
Gender: Male/Female classification
Chest Pain Type: Typical/Atypical/Non-anginal/Asymptomatic
Resting BP: Blood pressure at rest (mmHg)
Cholesterol: Serum cholesterol level (mg/dl)
Max Heart Rate: Maximum heart rate achieved
Fasting Blood Sugar: Normal/Elevated glucose levels
Resting ECG: Normal/ST-T Wave abnormality/LVH
Exercise Angina: Exercise-induced chest pain

CT Scan Analysis Model

Computed Tomography (CT) Scan Analysis

Best Model: Stacking Ensemble - 78.80% Accuracy
What is Cardiac CT?

Cardiac CT (Computed Tomography) uses X-rays and computer processing to create detailed cross-sectional images of the heart. It's particularly effective for detecting coronary artery calcification, measuring cardiac structures, and identifying anatomical changes associated with LVH, such as increased wall thickness and chamber enlargement.

Advanced CT Analysis Features
  • Cardiac Structure Analysis: Detailed anatomical measurements
  • Density Measurements: Tissue density quantification
  • Calcification Detection: Coronary artery calcium scoring
  • Texture Analysis: Myocardial tissue characterization
  • 3D Reconstruction: Volumetric cardiac modeling
  • Contrast Enhancement: Perfusion analysis
Advanced Ensemble Methods

Input Format

Clinical Significance

CT scans are particularly valuable for detecting coronary artery disease and calcification patterns that often accompany LVH, providing complementary information to other modalities.
