Overview

The LVH Detection API provides programmatic access to our advanced machine learning models for Left Ventricular Hypertrophy detection. The API supports multiple data modalities including:

  • ECG Signals: Time-series electrocardiogram data analysis
  • Medical Imaging: MRI and CT scan processing
  • Clinical Data: Patient demographics and vital signs
  • Multi-Modal: Combined analysis for enhanced accuracy

Base URL

http://localhost:5000

Authentication: Currently, no authentication is required for local development. For production deployment, implement appropriate security measures.

Available Endpoints

POST /predict

Submit patient data for LVH prediction analysis.

Request Parameters
Parameter Type Required Description
ecg_file File Optional ECG signal data (CSV format)
mri_file File Optional MRI image (PNG, JPG, DICOM)
ct_file File Optional CT scan image (PNG, JPG, DICOM)
age Integer Optional* Patient age (1-120)
sex Integer Optional* 0=Female, 1=Male
chest_pain_type Integer Optional* 0-3 (Typical to Asymptomatic)
resting_bp Float Optional* Resting blood pressure (mmHg)
cholesterol Float Optional* Cholesterol level (mg/dl)
* Either upload files OR provide clinical data (age, sex, chest_pain_type, resting_bp, cholesterol)

Code Examples

Python Example
import requests

# Health check
response = requests.get('http://localhost:5000/health')
print(response.json())

# Prediction with ECG file
files = {'ecg_file': open('patient_ecg.csv', 'rb')}
data = {
    'age': 65,
    'sex': 1,
    'chest_pain_type': 0,
    'resting_bp': 140,
    'cholesterol': 250
}

response = requests.post(
    'http://localhost:5000/predict',
    files=files,
    data=data
)

result = response.json()
print(f"Prediction: {result['prediction']}")
print(f"Confidence: {result['confidence_pct']}")
cURL Example
# Health check
curl http://localhost:5000/health

# Prediction with clinical data
curl -X POST http://localhost:5000/predict \
  -F "age=65" \
  -F "sex=1" \
  -F "chest_pain_type=0" \
  -F "resting_bp=140" \
  -F "cholesterol=250" \
  -F "ecg_file=@patient_ecg.csv"
