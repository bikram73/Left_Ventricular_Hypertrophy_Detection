Getting Started

What is LVH Detection System?

Our system uses artificial intelligence to detect Left Ventricular Hypertrophy (LVH) - a condition where the heart's main pumping chamber becomes thickened. The system can analyze ECG signals, MRI images, CT scans, and clinical data to provide accurate predictions.

System Requirements

  • Web Browser: Chrome, Firefox, Safari, or Edge (latest versions)
  • Internet Connection: Required for initial loading
  • File Formats Supported: CSV (ECG), PNG/JPG/DICOM (Images)
  • Maximum File Size: 16MB per file

Quick Start

  1. Navigate to the Home Page to understand system capabilities
  2. Go to Upload Page to submit your medical data
  3. View Results with detailed analysis and recommendations
  4. Check Documentation for technical details
  5. Use API for programmatic access
  6. Monitor System Health for status updates

Upload & Predict Guide

Smart Upload System

Our intelligent upload system accepts multiple types of medical data with flexible input options.

Two Input Options:
Option 1: File Upload
  • ECG Files: Upload CSV files with ECG signal data or features
  • MRI Images: Upload PNG, JPG, or DICOM cardiac MRI images
  • CT Scans: Upload PNG, JPG, or DICOM cardiac CT images
  • Optional Clinical Data: Fill additional clinical parameters
Option 2: Clinical Data Only
  • Age: Patient age (1-120 years)
  • Gender: Male or Female
  • Chest Pain Type: Typical, Atypical, Non-anginal, or Asymptomatic
  • Blood Pressure: Resting blood pressure (mmHg)
  • Cholesterol: Serum cholesterol level (mg/dl)
Step-by-Step Upload Process:
  1. Choose Input Method: Decide between file upload or clinical data entry
  2. Upload Files (if applicable): Drag and drop or click to select medical files
  3. Fill Clinical Form: Complete required fields (marked with *)
  4. Validate Input: System will show green checkmarks for valid inputs
  5. Submit Analysis: Click "Analyze LVH Risk" to start prediction
